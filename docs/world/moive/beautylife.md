# 《美丽人生》
### 1 为什么看这部电影？
理想爱情的模样
### 2 简介
- 背景:1937年的意大利，犹太青年圭多邂逅美丽的女教师多拉，生了一个可爱的儿子。不久后就纳粹就开始迫害犹太人，圭多和儿子都被送往集中营，多拉虽没有犹太血统，毅然同行，与丈夫儿子分开关押在一个集中营里。
- 导演：罗伯托·贝尼尼
- 补充：
    - 1933年起纳粹开始建立一系列集中营
    - 1939年第二次世界大战爆发后（持续到1945年），开始建立犹太区
    - 1941年至1945年，纳粹对欧洲诸种族及政治群体展开迫害，犹太人遭到广泛系统性屠杀，欧洲2/3的犹太人死亡，近600万人。
