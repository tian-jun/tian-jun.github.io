# 《寻梦环游记》
### 1 为什么看这部电影？
了解拉美文化的**亡灵节**，以便更好的阅读鲁尔福的小说《佩德罗·巴拉莫》
### 2 简介
- 墨西哥亡灵节：西班牙人来到美洲大陆后，他们把西方的“诸圣节”、土著的亡灵节以及土著的一些陪葬和祭祀风俗结合起来，创造了今天的亡灵节，日期定在11月1日和2日。如今在墨西哥和其它拉美国家，每年的11月1日是“幼灵”节，是怀念故去的孩子的节日，2日是“成灵”节，是大家缅怀故去的成年人的日子。  
印第安土著的亡灵节在每年七八月间。土著人认为，只有善待亡灵，让亡灵高高兴兴地回家过个节，来年活着的人才会得到亡灵保佑，无病无灾，庄稼也会大丰收。传统的纪念方式为搭建私人祭坛，摆放有糖骷髅、万寿菊和逝者生前喜爱的食物，并携带这些物品前往墓地祭奠逝者。  
### 3 电影片段
- 第一次获得万寿菊花瓣祝福语：“把我的照片放回到灵坛上，并且永远不许再碰音乐。”
- 第二次获得万寿菊花瓣祝福语：“永远，不要忘了我们一家人有多么爱你。”
- 第三次获得万寿菊花瓣祝福语：“我们把祝福送给你，米格，没有任何条件。”
- “这首歌不是写给世界的，是写给我的可可。”
- 米格用吉他弹唱Remember me，换醒了太奶奶对她的爸爸的记忆
