# 如何理解人与人之间的关系

人与人之间互动的本质因素有三：`交易，情感，信仰`。与他人的互动方式决定自己是什么样的人。如果交易，情感，信仰用不同的颜色表示，每个人都呈现出独一无二的色彩，由这三种颜色混合而成，颜色的不同就是震动的频率不同。每种因素的强烈程度用色彩的亮度表示，也就是光的振幅。这三种因素变化过程给人的感受，就是生命力，变化越剧烈，越感觉到强大的生命力。

### 交易
交易关系：要的是共赢，互不吃亏，交易对事不对人，不确定性低。
### 情感
- 情感对人不对事，
- 情感是自私的有独占性和唯一性。
- 亲密关系要的是时间，换句话说是陪伴，在事情上不分黑白，人是至高。
- 有感情就会有期待，这个是相生。在做不到无条件付出的时候就不要建立情感链接，可以先从对他好开始，但是不要有期待，一旦又了情感链接就会有了期待。
- 喜欢是一种源动力，就算没有期待，也可以喜欢一个人，喜欢程度和期待之间就是活动空间。
- 不能轻易的说喜欢，说太多了就不值钱，说喜欢之前，问一句自己，我为此付出求回报吗？
- 面对弱小的人，怜悯心态，面对同等实力的人，竞争心态，面对比自己强大的人，虔诚的心态。
- 遇到困难会告诉谁？不能承受这个困难的人不能告诉他，因为她如果爱你，又帮助不了你，她会难过。能承受这个困难的人，要看自己对他有没有安全感，有安全感就告诉他，没有安全感就只能自己一个人承担了。
- 绝望的时刻才知道什么是最重要的东西

### 信仰
- 在绝望的情景下仍然饱含希望，相信心中所追随的，从不改变的，就是信仰，越绝望，越坚定，越体现信仰。
- 要的是达成信仰，不在乎时间陪伴，以事情为主导，事情分黑白。

### 公众形象
公众形象的“他”不是他，他只是一个角色，一个扮演出来符合公众形象的角色。

### 男人
男人的友谊就是高山流水，势均力敌，男人向外求，战斗力。

### 女人
女人要的是，她是你世界的唯一，女人向内求，保护欲。

### 引用RGB颜色体系：
- red：信仰，橙红、黄色、棕色以及红色一端的色系常与炽热、温暖、热烈、热情有关，所以将其称为暖色调。
- green：情感
- blue：交易，蓝紫色系平静、安逸、通透、凉快相关联，就将其称为冷色调。
